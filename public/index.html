RESTful API for movie management and user authentication

Authentication

Most endpoints require a JWT Token. Login to receive your token and include it in the headers: Authorization: Bearer

Endpoints

POST /auth/login

Login to receive an Access Token.

Body: 
{
  "username": "your_user",
  "password": "your_password"
}
GET /movies

Returns a list of all movies in the database (Requires Token).

POST /auth/register

Create a new user account.
